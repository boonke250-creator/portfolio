# How to Add Your Sand Image and Music

## Quick Setup Guide

### Step 1: Copy Your Files to the Project Folder
1. Copy your sand image file to this folder (e.g., `sand.jpg`, `sand.png`)
2. Copy your music file to this folder (e.g., `background-music.mp3`, `ambient.mp3`)

Your project folder is: `c:\Users\Admin\Desktop\New folder\`

### Step 2: Update the HTML File
Once your files are in the folder, replace the placeholders in `index.html`:

**For the Background Image:**
- Find: `path/to/your/sand-background.jpg`
- Replace with: `sand.jpg` (or your actual filename)

**For the Background Music:**
- Find: `path/to/your/background-music.mp3`
- Replace with: `background-music.mp3` (or your actual filename)

### Step 3: Supported Formats
**Images:** `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
**Music:** `.mp3`, `.wav`, `.ogg`, `.m4a`

### Step 4: File Size Tips
- Keep images under 2MB for best performance
- Keep music files under 5MB
- Compress if needed
